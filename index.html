How to use kiwi slider

  1. Include kiwi.css

    <link rel="stylesheet" type="text/css" href="stylesheets/kiwi.css">

  2. Include jQuery and kiwi-slide

    <script src="//code.jquery.com/jquery-latest.min.js"></script>
<script src="javascripts/kiwi-slider.js"></script>
  3. new KiwiSlider('kiwi-slider');

    There are some options below

options

{ 
    trigger: 'click', //click or mouseover
    direction: 'H'  //Horizontal or Vertical H is default
}

kiwi-slider also support mobile device

You can visit http://loethen.github.io/kiwi-slider/ on your mobile browser or change to chrome device mode on PC
